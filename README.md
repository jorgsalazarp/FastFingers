###### FAST FINGERS

# El proyecto se trata de un pequeño juego de mecanografía para Android desarrollado en Kotlin.
# Tiene una premisa bastante simple: Hacer desaparecer palabras escribiendolas antes que toquen el suelo

## CARACTERISTICAS PRINCIPALES

# MECANICA: Palabras aleatorias comenzarán a caer a distintas velocidades. El juego detecta *La palabra que mas abajo se encuentre*
# BASE DE DATOS INTERNA: Room, almacena 100 palabras.
# API Rest: Guarda puntajes en una API junto con el nombre del usuario que logró puntajes, se muestran en un ranking.

## INSTALACION

1. Clonar repositorio
2. Abrir proyecto en Android Studio
3. Sincronizar proyecto con Gradle
4. Conectar dispositivo Android mediante USB o utilizar el emulador (Consume muchos recursos)
5. Darle click a RUN
